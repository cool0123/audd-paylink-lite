# X Post Draft

Built a quick MVP for `AUDD PayLink Lite` — a simple merchant payment-link and settlement dashboard concept for AUDD on Solana.

What it demonstrates:
- Create AUDD-denominated payment links
- Preview a checkout page + QR-style payment view
- Track pending / paid / expired states
- Export settlement CSV rows for merchants

This is a safe static demo: no private keys, no wallet signing, no live payment execution.

The production path is clear: official AUDD mint config → wallet adapter checkout → on-chain transfer verification → merchant settlement tools.
